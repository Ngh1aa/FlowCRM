# FlowCRM - Business Analysis Documents

This folder contains Business Analysis artifacts for the FlowCRM project.

## Documents

### 1. Business Requirements Document (BRD)
**File:** `BRD.md`

Covers:
- Problem statement and business context
- Project goals and success criteria
- Scope and boundaries (MVP)
- Stakeholder analysis
- Risk assessment

### 2. User Stories & Acceptance Criteria
**File:** `UserStories.md`

Covers:
- Persona definitions (Sales Rep, Sales Manager)
- Detailed user stories with acceptance criteria
- Non-functional requirements
- Sprint planning hints

### 3. Process Flow Diagram
**File:** `ProcessFlow.md`

Covers:
- Lead to Deal lifecycle activity diagram
- Deal stages and transitions
- Touchpoints and handoffs
- System integrations

### 4. Use Case Diagram
**File:** `UseCase.md`

Covers:
- Actor definitions
- Use case descriptions
- Relationship mappings
- System boundaries

## Quick Reference

### Deal Stages
1. **Chào hàng (Qualification)** - Initial contact, qualifying leads
2. **Đàm phán (Proposal)** - Active negotiation
3. **Đề xuất (Negotiation)** - Proposal sent, awaiting response
4. **Chốt (Closed Won)** - Deal won
5. **Thất bại (Closed Lost)** - Deal lost

### KPIs
- **Deals đang mở**: Target 24+ active deals
- **Doanh thu dự kiến**: Target ₫2.4B+ pipeline value
- **Tỷ lệ chuyển đổi**: Target 35%
- **Khách hàng mới**: Target 8+ new customers/month
