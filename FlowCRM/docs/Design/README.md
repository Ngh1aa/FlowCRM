# FlowCRM - Design Documents

This folder contains UI/UX Design artifacts for the FlowCRM project.

## Documents

### 1. Design System
**File:** `DesignSystem.md`

Covers:
- Color palette with hex codes
- Typography scale (Inter font family)
- Spacing system
- Border radius & shadows
- Component inventory

### 2. Screen Specifications
**File:** `Screens.md`

Covers:
- Dashboard layout (Bento Grid)
- Pipeline Kanban structure
- Customer List (Spreadsheet-like)
- Quick Create Modal specs

---

## Design Principles

1. **Bento Grid Layout** - Organized, scannable dashboard
2. **Spreadsheet UI** - Familiar data table patterns
3. **Visual Hierarchy** - Important info at top
4. **Drag & Drop** - Intuitive task completion
5. **Real-time Feedback** - Instant validation, updates
